## 第三章：生意本质分析 — 段永平"对的生意"

### 这门生意的本质

> **ASMPT是半导体和电子制造领域的"高端精密设备制造平台"——客户买的是"能把芯片加工得更小、更快、更密的机器"，而不是芯片本身。**

### 商业模式画布

| 维度 | 描述 |
|------|------|
| **客户** | 晶圆厂（IDM：Intel、Samsung、SK海力士）、封测厂（OSAT：ASE、Amkor、JCET）、EMS代工厂（富士康等） |
| **产品** | 高精度半导体封装设备（TCB、HB、Flip Chip、Wire Bonder）、SMT贴片机 |
| **收入模式** | **一次性设备销售为主**（~85%），辅以备件和服务（~15%） |
| **复购驱动** | 技术迭代周期（每2-3年新设备需求）+ 产能扩张周期 |
| **客户粘性** | 中高——认证周期长（6-18个月），一旦成为工艺中的"工艺记录"很难被替换 |
| **周期性** | 强——与半导体资本开支周期高度相关 |

### 盈利趋势深度分析

**5年收入与利润趋势：**

```
2021: HK$219亿 / EPS HK$7.72（疫情驱动的超级周期峰值）
2022: HK$194亿 / EPS HK$6.36（周期下行）
2023: HK$147亿 / EPS HK$1.73（行业去库存低谷）
2024: HK$125亿 / EPS HK$0.71（周期底部——5年最低盈利）
2025: HK$137亿 / EPS HK$2.61（复苏期，AI驱动）
```

**关键观察：**
1. **行业强周期性的教科书级案例**：收入从2021年峰值到2024年低谷下降**43%**，利润下降**91%**
2. **2025年复苏的性质**：主要由AI驱动的先进封装（TCB收入+146% YoY）拉动，而非全行业复苏
3. **毛利率趋势暗藏隐忧**：2021年40.6% → 2025年37.8%，下降280bps，主要因SMT低毛利业务占比提升
4. **经营杠杆正在正向转换**：2025年收入+10%，调整后经营利润+22%——表明固定成本基数已压缩到位

### 经营杠杆分析

| 指标 | FY2023 | FY2024 | FY2025 |
|------|--------|--------|--------|
| 毛利率 | 39.3% | 40.0% | 37.8% |
| 研发费用率 | 13.9% | 15.4% | 14.0% |
| 销售管理费用率 | 17.8% | 20.0% | 19.2% |
| 调整后经营利润率 | 4.3% | 4.6% | 5.1% |
| 调整后净利率 | 3.3% | 3.0% | 3.4% |

观察：2024-2025年收入仅恢复12%，但调整后净利润增长24%，说明经营杠杆在收入回升时体现明显。

### 段永平式追问

> **"这门生意好在哪？如果只能用一句话描述，是什么？"**

好在这是一门"卖铲子"的生意——AI浪潮中所有芯片厂商都在扩产，不论谁赢，都需要ASMPT的设备来封装芯片。但坏处是：这是一次性设备销售，客户买了之后2-3年才可能再买，且周期性极强。**不是复购驱动的"好生意"，而是在正确周期能赚大钱的"周期性好生意"。**

## 第四章：护城河评估 — 巴菲特"经济护城河"

### 五类护城河逐一验证

#### 1. 技术/专利壁垒 ⭐⭐⭐⭐（最核心）

| 维度 | 评估 |
|------|------|
| **TCB技术** | ASMPT是热压键合（TCB）市场的绝对领导者，市占率目标35-40%。TCB是HBM（高带宽内存）封装的关键工艺，用于NVIDIA、AMD的AI芯片。 |
| **等离子AOR技术** | 自主研发的等离子辅助AOR技术是超细间距C2W键合的核心壁垒，竞争对手难以复制。 |
| **全产品线覆盖** | 唯一一家提供从Wire Bonder到TCB到Hybrid Bonding全系列后端封装设备的公司。 |
| **专利护城** | 数十年积累的封装设备专利群，新进入者需要5-10年才能建立同等水平。 |

#### 2. 转换成本 ⭐⭐⭐

| 维度 | 评估 |
|------|------|
| **认证周期** | 设备进入客户产线需6-18个月的认证，一旦成为"工艺记录"很难被替换 |
| **但** | 每个新世代产品都是重新竞争的机会——客户可以在新产线选择竞品 |
| **软件锁定** | 自研MES软件（Critical Manufacturing）和机器健康管理系统增加了一定粘性 |
| **备件与服务** | 设备装机后，备件和服务的持续收入形成自然锁定 |

#### 3. 品牌/定价权 ⭐⭐

| 维度 | 评估 |
|------|------|
| **品牌溢价** | 在封装设备领域品牌认知度高，但并非消费者品牌 |
| **定价权** | 取决于技术独特性——TCB有较强定价权（技术领先），Wire Bonder和SMT定价权较弱（竞争激烈） |
| **毛利率变化** | 2021-2025年毛利率下降280bps，说明整体定价权在削弱，但2026Q1 SEMI毛利率回升至46.4%显示产品组合改善 |

#### 4. 规模效应 ⭐⭐⭐

| 维度 | 评估 |
|------|------|
| **全球最大后端设备商** | 后端封装设备市占率~30%，SMT设备市占率~22%（来源：DBS） |
| **供应链议价能力** | 大规模采购零部件有成本优势 |
| **R&D分摊** | 年研发投入HK$19亿，摊薄到大规模收入上仍有竞争力 |

#### 5. 网络效应 ⭐（极弱）

| 维度 | 评估 |
|------|------|
| 用户越多产品越好 | 几乎不存在——设备商不依赖网络效应 |

### 巴菲特式追问

> **"10年后这条护城河还在吗？什么能摧毁它？"**

10年后ASMPT的护城河大概率还在，但面临以下威胁：
1. **技术路线变更**：如果行业从TCB转向Hybrid Bonding，ASMPT的领先优势可能被削弱——面临Applied Materials、Tokyo Electron等前道设备巨头的竞争
2. **中国竞争**：中国本土封装设备商（如中微公司、盛美上海）正在快速追赶，可能在5-10年后在中低端市场构成实质性竞争
3. **客户垂直整合**：大客户（如Intel、Samsung）可能自研部分封装设备

### 空方核心论点

1. **估值过高**：P/E（TTM）高达100x，即使按正常化EPS计算也在73x——远超行业平均，已将未来3-5年的乐观预期完全定价
2. **周期性风险被低估**：市场把ASMPT当成AI成长股，但它本质是周期性资本品——2021年HK$130到2024年HK$55的跌幅就是前车之鉴
3. **TCB竞争的"隐形天花板"**：Besi在C2W领域竞争激烈，Disco在划片/研磨领域有独特优势，前道巨头（AMAT/TEL）正向后端延伸
4. **SMT业务是战略负资产**：毛利率仅~27%，竞争激烈，剥离如果失败可能长期拖累利润率

### 芒格式追问

> **"我最可能在哪里犯错？"**

最可能的错误是**低估了周期风险**。当前ASMPT的AI叙事极其完美：TCB +146%年增长、HBM4订单锁定、Photonics五倍增长。但半导体行业从未有"这次不一样"的线性外推——每次高峰时看起来都像是永续增长。当AI资本开支在2027-2028年进入下一轮调整期时，ASMPT的订单可能骤降30-50%，届时100x PE的估值悬崖将是毁灭性的。

> **"聪明人为什么不买/做空这家公司？"**

聪明人不买的原因很可能就是**估值和周期位置**——在周期复苏早期买入（2025年初，股价~HK$70-80）是对的，但在股价已涨近4倍（从52周低点HK$55.30算起）后追入，安全边际已非常薄。

### AI驱动的"文明级范式转移"

**判断：AI是文明级范式转移，ASMPT处于受益链的关键环节。**

关键变化在于：传统上封装是半导体价值链中附加值最低的环节，但AI正在**将价值和复杂度前所未有地向后端封装倾斜**——先进封装（TCB、HB）的精度要求甚至高于部分前道工艺。

## 附录三：TCB技术详解

TCB（Thermal Compression Bonding，热压键合）是先进封装中的关键工艺：
- **作用**：将芯片精确键合到基板或另一个芯片上，在加热和加压下完成互连
- **应用场景**：HBM内存堆叠、AI逻辑芯片的2.5D/3D封装
- **ASMPT优势**：拥有等离子辅助AOR（Advanced Oxidation Removal）技术，可以实现亚微米级键合精度
- **竞争格局**：ASMPT市占率~35-40%，主要对手Besi（荷兰，C2W强项）、Disco（日本，划片/研磨）
- **技术迭代**：Chip-to-Substrate（C2S）→ Chip-to-Wafer（C2W）→ Hybrid Bonding（HB）
- **价值量提升**：每个技术代际的设备ASP提升30-50%
